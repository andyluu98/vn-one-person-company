# Templates Custom (BYOT)

Đặt template riêng của DN vào đúng folder con theo **mã phòng chính thức** (xem `departments/` trong repo):

```
00-Templates-Custom/
├── 04-people/              # JD, hợp đồng LĐ, sổ tay NV
├── 03-finance/             # phiếu thu chi, hoá đơn
├── 05-operations/          # SOP, biên bản
└── 07-marketing/           # brief, kế hoạch QC
```

Mã phòng đầy đủ: 01-governance, 02-strategy, 03-finance, 04-people, 05-operations, 06-sales, 07-marketing, 08-customer, 09-product-tech, 10-training, 11-reporting, 12-growth

Hệ thống sẽ ưu tiên template ở đây hơn template mặc định.
Format hỗ trợ: .md, .docx, .xlsx
